# Procedure:
1. install eclipse gradle plug-in:
help -> eclipse marketplace -> search gradle
2. install gradle on mac:
brew install gradle
3. go to gradle project directory:
gradle build
gradle eclipse
4. import existing workplace project into eclipse and run
